# CSS-Mastery-Portfolio

A modern frontend portfolio project built to demonstrate **advanced CSS concepts** combined with **interactive JavaScript features**.  
This project showcases layout systems, animations, design systems, and DOM interactivity — structured like a real-world UI project.

---

##  Project Overview

**CSS Mastery Portfolio** is a learning-focused yet production-style frontend project covering:

- CSS Grid
- Flexbox
- CSS Variables (Design System)
- Animations & Transitions
- Advanced Selectors
- BEM Methodology
- Performance Optimization
- JavaScript Interactivity

It demonstrates how styling and behavior work together to create a smooth user experience.

---

##  Technologies Used

| Technology | Purpose |

###  JavaScript Features

| Feature | Description |
|--------|-------------|
|  Dark Mode Toggle | Switch between light & dark themes |
|  Theme Memory | Saves theme using localStorage |

|  Scroll Reveal | Cards animate into view while scrolling |
